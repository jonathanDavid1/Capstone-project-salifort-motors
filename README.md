### 📌 Descripción del Proyecto
Este proyecto representa la culminación del **Google Advanced Data Analytics Professional Certificate**. El objetivo es proporcionar insights estratégicos al departamento de Recursos Humanos de **Salifort Motors** para predecir si un empleado dejará la empresa, utilizando un enfoque de Ciencia de Datos riguroso.

### 🎯 Objetivos Técnicos
- **EDA (Exploratory Data Analysis):** Identificación de correlaciones clave entre carga de trabajo, evaluaciones y satisfacción.
- **Modelado Predictivo:** Implementación de modelos para clasificar el riesgo de salida de los empleados.
- **Optimización de Retención:** Generación de sugerencias basadas en datos para reducir el *turnover*.

### 🚀 Metodología PACE
He estructurado este laboratorio siguiendo las fases del marco de trabajo **PACE**:

1.  **P (Plan):** Definición del problema y análisis de las variables del dataset.
2.  **A (Analyze):** Limpieza de datos y visualización con `Seaborn` y `Matplotlib`.
3.  **C (Construct):** Ingeniería de características y entrenamiento de modelos de clasificación.
4.  **E (Execute):** Evaluación de métricas de desempeño (F1-score, Recall) y entrega de recomendaciones.

### 🧪 Stack Tecnológico
- **Análisis de Datos:** Pandas, NumPy.
- **Visualización:** Matplotlib, Seaborn.
- **Machine Learning:** Logistic Regression / Random Forest / XGBoost (según elección en el lab).
- **Entorno:** Jupyter Notebook / Google Colab.

### 💡 Conclusiones del Análisis
El modelo permite identificar patrones de agotamiento (burnout) y desequilibrio en la carga de proyectos, proporcionando una herramienta preventiva para la toma de decisiones en RR.HH.
